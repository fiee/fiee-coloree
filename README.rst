fiëé colorée
============

What and how
------------

* Widgets and tools for RGB and CMYK colors


Dependencies
------------

* Django 1.6 with included contributions
* widgets use jQuery_ and jquery.colorPicker


Known Issues
------------

* No tests
* No docs
* No color management
